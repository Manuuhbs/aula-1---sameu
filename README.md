## realizando commits e enviando ao repositório

## configurar o Git
pode ser via web diretamente ou via prompt através de:
git config --global user.name "nome"
git config --global user.email "nome@email.com"

## criar commits
git add .
git commit -m "mensagem"

## enviar para o Github
após dar um create e pegar o link do repositorio, deve-se ir no terminal e colocar git remote add origin https://github.com/Manuuhbs/el-gato-de-buetas.git
git push -u origin master

## pequeno bonus
git status - confere mudanças
